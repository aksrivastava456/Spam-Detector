# Spam Detector

A full-stack machine learning web application that detects whether a message is Spam or Not Spam using TF-IDF and Multinomial Naive Bayes.

## Tech Stack
- React
- Flask
- Scikit-learn
- TF-IDF
- Multinomial Naive Bayes
